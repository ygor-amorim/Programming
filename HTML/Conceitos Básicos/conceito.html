Um documento HTML é formado por textos simples estruturados com ELEMENTOS: Elementos -> Parte de uma página web, pode conter um item de dados, ou um bloco de texto, ou uma imagem, ou até mesmo nada. Um elemento típico possui tag de abertura com alguns atributos, o conteúdo de texto incluido e uma tag de fechamento. isso é um exemplo de elemento Elementos e Tags não são a mesma coisa, as tags começam ou terminam um elemento no código fonte, enquanto elementos são parte do DOM, o modelo do documento para exibir a página no navegador. Como demonstrado anteriormente, elementos são acompanhados de abertura e fechamento de tags. Cada tag começa e termina com colchetes angulares (<>). Existem algumas tags vazias/sem conteudo que não podem incluir qualquer texto, como por exemplo a tag "" Arquivos HTML sao salvos como uma extensao .htm ou .html, é disponibilizado por um servidor web e pode ser renderizado por qualquer navegador.
