Primeiramente, HTML (HyperText Markup Language) não é uma linguagem de programação, mas sim uma linguagem de *MARCAÇÃO*, isto é, ela serve para estruturar um documento em seções lógicas e incorpora conteúdos como imagem e vídeos em uma página

Este texto está dentro de um parágrafo, graças as tags de marcação de elemento que estão no início e no fim desde elemento.

Anatomia de um elemento

As principais partes de um elemento são:

1. Tag de abertura: Consiste no nome do elemento (neste caso, p), envolvido de parenteses angulares de abertura e fechamento. Isso indica onde o elemento começa, ou inicia a produzir um efeito - neste caso, onde o paragrafo se inicia.

2. Tag de fechamento: é o mesmo que a tag de abertura, exceto que este inclui uma barra diagonal antes do nome do elemento. indica onde o elemento termina - neste caso, onde fica o fim do parágrafo.

3. Conteúdo do elemento, neste caso é somente texto.

4. O elemento: A tag de abertura, mais a tag de fechamento, mais o conteúdo, isso tudo junto resulta no elemento.

Exemplo:

Para italizar um texto, é preciso utilizar a tag de abertura e fechamento invocando o nome do elemento, no caso seria "em", e incluir o conteúdo desejado entre as tags.

Assim fica tudo italizadinho!

Aninhando elementos

Elementos podem ser inseridos em outros elementos, isto é chamado de aninhamento. Mas é importante aninhar de forma correta, ao abrir um elemento dentro do outro, devemos fechá-lo antes de fechar o primeiro, pois além de tornar a marcaçao mais organizada, permite que o navegador compreenda sem erros ou confusões.

Funciona basicamente como regras utilizando expressoes com chaves, colchetes e parenteses na matemática, se eu tenho (a - [c+b]), podemos observar que o colchete nao pode ser fechado após os parenteses, pois ele esta inserido nos mesmos.

Exemplo:

Meu parágrafo é muitissimo interessante, nele eu posso escrever o que eu quiser!, porém, devo me atentar para ter todos meus elementos bem organizados, e aninhá-los corretamente

Elementos vazios. Nem todo elemento segue o padrao acima, alguns consistem apenas em uma unica tag, utilizada para inserir/incorporar algo no doc no lugar em que ele é incluido. Por ex, o elemento insere uma imagem em uma pagina na posiçao em que ele é incluido. Exemplo: Elementos vazios também sao conhecidos como void elements.
